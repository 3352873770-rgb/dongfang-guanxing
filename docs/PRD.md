# 东方观星一级首页 PRD

## 产品目标

为对《周易》、东方哲学、自我探索和决策辅助感兴趣的用户，提供一个可信、克制、可理解的一级入口。首页负责建立氛围、解释价值、展示可用玩法，并将用户引导至后续问卦或工具流程。

## 核心用户需求

1. 在情感、事业、学业、财富等不确定情境中获得结构化思考入口。
2. 快速理解不同问卦和术数工具的用途，而不是面对晦涩术语。
3. 感受到传统文化氛围，同时保有理性、不迷信的安全边界。
4. 在手机上顺畅浏览和选择，不被复杂特效阻碍。

## 当前范围

- 首页 Hero、昼夜模式与品牌叙事
- 长期问卦和灵签入口
- 今日一卦
- 人格图谱入口
- 六十四卦图谱预览
- 六十四卦知识二级页：基础阅读方法、当前卦象知识与完整图谱检索
- 推荐小工具
- 问卦记录价值说明
- 理性使用提示

## 不在当前范围

- 真实付费、登录和账户系统
- 完整问卦算法与 AI 报告生成
- 医疗、法律、投资等高风险决策替代
- 除六十四卦知识页之外的二级页面完整业务闭环

## 成功标准

- 390px 及以上宽度无横向溢出。
- 首页图谱预览不使用水平滚动，卦象入口可进入知识二级页。
- 昼夜模式内容、结构和核心交互一致。
- 核心按钮可通过鼠标、键盘和触摸使用。
- 动效不影响内容对比度，并支持 reduced-motion。
- 生产构建和自动检查通过后才能发布。
