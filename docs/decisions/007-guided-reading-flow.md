# 007 开始观星问卦引导流程

- 状态：已完成本地实现与浏览器验证
- 日期：2026-07-24
- 关联 Issue：#17

## 决定

“开始观星问卦”使用同一个全屏连续流程，不在步骤之间跳回首页：

```text
选择所问方向
    ↓
写下一个具体问题
    ↓
选择或新建人物档案
    ↓
确认所问、档案与起卦方式
    ↓
进入后续排卦环节
```

首页 Hero 的“开始观星问卦”从第一步进入。“长期问卦”中的感情发展、事业发展、学业考试和财富运势按钮可预选对应方向，直接进入问题输入。返回上一步时保留已选择的方向、问题和档案信息。

四个流程状态的页眉统一使用“·观星问卦 ·”，不显示步骤或阶段提示。

## 首期分类

- 感情发展
- 事业发展
- 学业考试
- 财富运势
- 其他所问

问题输入必须围绕一件具体事情。界面提示用户补充处境和希望看清的变化，不将问卦包装为医疗、法律、投资或现实决策的替代。

## 起卦准备

当前提供“三枚铜钱”和“按时起卦”两个入口，只完成起卦前的信息确认与入口状态。真实六爻投掷、时间排卦、卦象生成和解读属于下一阶段，不在本次范围内。
