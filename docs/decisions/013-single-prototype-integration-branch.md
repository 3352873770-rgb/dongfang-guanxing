# 013 使用唯一原型集成分支

- 状态：已确认，本地验证通过，待发布确认
- 日期：2026-07-24
- 关联 Issue：#29

## 决定

`agent/prototype-current` 是当前未发布产品的唯一集成分支。

```text
agent/prototype-current
    ├── agent/<feature-a>
    ├── agent/<feature-b>
    └── agent/<feature-c>
```

每个功能从集成分支创建独立任务分支，在自己的分支完成设计、开发和验证；验收后合回 `agent/prototype-current`。后续功能继续从更新后的集成分支开始，不再长期采用“功能分支叠功能分支”的链式结构。

## 当前集成内容

- 观星问卦单页流程与共享人物档案
- 五个灵签信二级工具
- 六十四卦知识页
- 今日卦象两页式流程
- 五个灵签工具的人物档案置顶与记录关联

三枚铜钱和人格探索仍处于独立开发状态，完成验证前不进入集成分支。

## 发布边界

`main` 继续作为公网稳定分支。集成分支只有在用户针对当前发布任务明确确认后才能合入 `main`；建立或更新集成分支本身不触发 GitHub Pages 发布。
