# Decision 016：人格偏好探索使用单页五点量表与同页结果

- 日期：2026-07-24
- 状态：已合入 `agent/prototype-current`，待发布确认
- 关联：Issue #28
- 视觉源：`docs/design/personality-preference-flow-v1.png`

## 决定

人格入口进入 `#/personality` 独立二级页。页面沿用共享二级页页眉和昼夜切换，Banner 中心展示大标题，LightRays / LiquidEther 仅在 Banner 可见时运行。

12 道题在一个可纵向滚动的语义化表单中完成。每道题第一行显示问题，第二行以“两端文本 + 中间五点”表达从强烈偏向左端到中立、再到强烈偏向右端的连续倾向。

一次提交后，在同一路由继续呈现人格偏好卡片、四轴结果、可选核对题和五段人格介绍，不增加步骤页或额外结果路由。

首页人格入口同步使用 `public/media/legacy/personality-preference-v2.webp`，以四轴印谱替换旧八维雷达，避免入口承诺与实际结果模型不一致。

## 原因

- 保持用户对完整题量和进度的掌控，减少反复翻页与上下文中断。
- 五点量表比二选一更能表达倾向强度与中立状态。
- 结果卡先给出清晰结论，连续介绍再解释其含义，符合先识别、后理解的阅读顺序。
- 明确非诊断边界，避免把简化问卷误解为官方 MBTI 或确定性人格判定。

## 回滚

移除 `#/personality` 路由与首页人格入口事件，删除人格页面、数据、样式和对应文档即可；不会影响首页、问卦、日签或六十四卦知识页。
