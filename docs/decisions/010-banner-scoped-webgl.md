# 010：Banner 范围内按需挂载 WebGL

- 状态：已采纳
- 日期：2026-07-24

## 背景

首页与六十四卦知识页需要保留昼夜氛围，但全页持续运行 LightRays 或 LiquidEther 会在用户阅读下方内容时继续占用 GPU，知识页的固定画布也会把动态视觉带入不需要动效的图谱区。

## 决定

两类页面都把 WebGL 画布限制在独立 Banner 容器内，并通过共享可见性 Hook 同时监听 Banner 与文档状态。只有 Banner 位于视口且页面位于前台时才挂载 LightRays 或 LiquidEther；离开范围或标签页进入后台后立即卸载。

Banner 以下继续保留对应昼夜主题的静态背景、色彩和对比关系，不运行 WebGL。系统 reduced-motion 偏好仍优先关闭持续动画。

## 影响

- 用户阅读长期问卦、工具和六十四卦正文时不再持续占用 WebGL。
- 动效的视觉边界与信息结构一致，避免光线或流体穿透到正文和图谱。
- 昼夜切换、Banner 构图和原有交互不变。
- 本任务仍停在本地验证与草稿 PR，等待用户单独确认公网发布。
