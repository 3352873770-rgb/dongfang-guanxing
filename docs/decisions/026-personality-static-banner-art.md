# 026 人格页使用静态 Banner 背景

## 决策

人格偏好探索页不再挂载 LightRays 或 LiquidEther。Banner 使用专用静态图片 `public/media/personality/personality-banner-ink-landscape-v1.jpg`，日间显示暖米纸与淡墨山水，夜间通过 CSS 滤镜转为低亮深墨。

该决策取代 016 中关于人格 Banner 使用 WebGL 动态氛围的约定；首页与六十四卦知识页的既有动态背景不受影响。

## 原因

人格问卷属于连续阅读和选择任务。动态光线与流体会抢夺标题和问卷注意力，也使 Banner 呈现偏蓝色块，与下方墨纸正文割裂。静态水墨图能保留产品气质，同时降低视觉噪音与 GPU 开销。

## 图片要求

- 中央保持低细节留白，保证大标题可读。
- 山水只分布在底部和边缘，不出现人物、建筑、八卦、星座、文字或水印。
- 昼夜复用同一文件，不维护两套图片。
- 图片按 Banner 比例压缩，避免重新引入首屏体积问题。
