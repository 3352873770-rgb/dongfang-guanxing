# 022 三枚铜钱返回首页根地址

- 状态：已采纳
- 日期：2026-07-24

## 决定

三枚铜钱二级页的共享页眉退出操作使用空 Hash 和“返回首页”文案。共享页眉遇到空 Hash 时通过当前 `pathname` 明确导航到站点根路径；用户退出后回到首页顶部，地址不保留 `#tools`、空 `#` 或查询参数。

其他二级页的上下文返回规则不受影响。

## 原因

`#tools` 本质上只是首页工具区锚点，但它会改变浏览器地址。用户可能把根地址和带锚点地址误解为两个不同网站。三枚铜钱退出后回到首页顶部，可以让站点身份和返回结果更加明确。

## 回滚

如需恢复工具区上下文返回，将三枚铜钱页的 `backHash` 改回 `tools`、文案改回“返回工具”，并重新验证 URL、滚动位置和浏览器前进/后退行为。
