# 020 使用 MMEETT 官方 Logo 路径

- 状态：已采纳
- 日期：2026-07-24

## 决定

以用户提供的 `mmeett logo最新版(1).pdf` 为品牌视觉依据。首页导航、Hero 和全部二级页页眉统一复用 `src/brand-lockup.jsx` 中的 MMEETT 图形标和字标 SVG 路径。

`MMEETT Fate` 继续作为产品名称出现在静态标题、页面标题与说明文案中；官方 Logo 的可视字标只显示 PDF 中确认的 `MMEETT`，不再把 `Fate` 伪造成同一套字形。

## 原因

此前的 Cormorant Garamond 文字组合并非官方字标，字体渲染会随设备变化且与提供的几何品牌稿不一致。使用矢量路径可以保持比例、笔画和昼夜反白的一致性，同时不额外加载字体资源。

## 影响与回退

移除 `@fontsource/cormorant-garamond` 依赖，降低首屏字体资源。若需回退，恢复共享组件的文字字标、字体依赖和相关样式即可；不影响路由、昼夜主题或业务流程。
