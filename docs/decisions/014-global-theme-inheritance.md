# 014 全站继承同一昼夜主题

- 状态：已确认，本地验证通过，待发布确认
- 日期：2026-07-24
- 关联 Issue：#32

## 决定

首页是全站昼夜主题的唯一选择入口。用户在首页选择昼或夜后，进入观星问卦、灵签工具、今日卦象、六十四卦知识页或后续二级功能时，必须继续呈现同一主题。

主题切换不复制页面、路由、组件或业务状态。所有页面共用根节点的 `data-dfgx-theme` 与 `dfgx-theme` 本地偏好；页面只消费共享设计变量。

```text
首页主题状态
    └── data-dfgx-theme
          ├── 首页背景氛围
          ├── 问卦与灵签共享表单
          ├── 今日卦象
          └── 六十四卦知识页
```

## 实现边界

- 昼间保持暖米纸背景、深墨文字和旧金控件。
- 夜间使用深墨背景、浅米文字和低透明金边。
- 布局、内容、档案组件、表单逻辑和结果逻辑保持完全共用。
- 只允许为背景氛围和可读性调整必要的颜色、边框及表面变量，不得为主题维护两套页面。

## 验证

在 `http://127.0.0.1:4175/` 分别从昼、夜首页进入观星问卦与云签解惑，根主题、页面背景、宣纸容器、输入控件和文字对比均保持一致。今日卦象与六十四卦知识页继续沿用相同主题状态。
