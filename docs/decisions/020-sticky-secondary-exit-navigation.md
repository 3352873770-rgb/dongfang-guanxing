# Decision 020：长内容二级页使用固定退出导航

日期：2026-07-24

## 决策

人格偏好、今日卦象、六十四卦知识和三枚铜钱等长内容二级页统一复用 `SecondaryPageHeader`。页眉使用粘性定位，在用户纵向滚动时持续位于视口顶部，并始终显示返回首页或返回工具的退出操作、品牌入口和昼夜切换。

## 原因

这些页面采用连续纵向阅读和单页任务流。原页眉只在页面顶部出现，用户进入中段后无法快速退出，需要反向滚动较长距离。固定页眉能保留当前单页结构，同时提供稳定的全局出口与主题控制。

## 约束

- 返回操作保持至少 44px 触控高度，并通过可见箭头、文字和可访问名称表达“退出当前页面”。
- 移动端避让安全区，不产生横向溢出或遮挡主要内容。
- 共享页眉必须位于 Banner 裁切容器之外，避免 `overflow` 或 `contain` 限制粘性范围。
- 返回首页后定位到对应一级入口；不删除或重置当前设备的主题偏好。
- 新增长内容二级页优先复用同一组件，不复制固定导航实现。
