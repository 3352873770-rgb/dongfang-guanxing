# 025 背景色连续性

## 决策

首页 Hero、Legacy 内容区与人格偏好二级页在昼夜模式中分别共享同一底色家族。层级差异依靠光照、低透明度纸张纹理和渐变过渡建立，不再依靠明显偏色或亮度突变的整块背景。

人格偏好页缩短移动端 Banner 高度并降低正文纹理透明度；Banner 底部淡入页面底色，问卷说明、端点文字和中立标签提高对比度。

## 原因

原实现中，日间 Hero 偏冷灰、下方内容偏暖米纸；人格页 Banner 与问卷区也存在明显色块和大段空白。单个模块成立，但连续阅读时像不同设计稿拼接。

## 影响

- 不改变页面结构、路由、业务逻辑或昼夜主题状态。
- 不新增插画或独立背景资产。
- 保留现有 LightRays、LiquidEther、宣纸与墨纸素材。
- 后续二级页应优先复用主题底色，再通过局部表面层建立层级。
