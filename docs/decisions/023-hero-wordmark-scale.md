# 023 首页 Hero 字标尺度

- 状态：已采纳
- 日期：2026-07-24

## 决定

首页中心 MMEETT 官方字标在桌面端使用 `clamp(420px, 42vw, 860px)`，使其成为首屏唯一主视觉。720px 以下使用 `min(86vw, 360px)`，保持单行、居中与无横向溢出。

## 原因

用户提供的案例只作为字号比例参考：现有 `430px` 上限压低了桌面品牌存在感。放大现有官方 SVG 字标可以提高首屏识别度，同时不采用案例的字体、`Fate` 字形或布局。

## 回滚

恢复 Hero 字标的两个宽度 token 即可，不影响导航、昼夜主题、Logo SVG 路径或业务流程。
