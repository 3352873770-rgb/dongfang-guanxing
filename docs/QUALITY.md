# Quality Baseline

## 本地门禁

提交前执行：

```bash
npm run check
```

检查内容：

- Node 内置测试验证关键文件、页面入口和设计交互契约。
- Vite 生产构建验证依赖与资源路径。
- GitHub Pages 基础路径构建验证公网资源引用。

## 手动视觉回归

视觉或交互修改额外检查：

- 桌面端 1440×1024。
- 移动端 390×844 和最小 320px。
- 昼模式、夜模式与切换后刷新。
- `#ask`、`#tools` 每个按钮独立 hover/focus/active。
- reduced-motion。
- 浏览器控制台无新增错误。

## 通过标准

- `npm run check` 全部通过。
- 无未解释的构建警告或新增控制台错误。
- 公网资源路径不指向本机或站点根目录错误位置。
- PR 中记录验证结果与回滚方式。
