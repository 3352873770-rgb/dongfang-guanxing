# Changelog

本项目采用语义化版本。用户可感知、架构、发布和项目治理变化均记录在此。

## Unreleased

- 将首页六十四卦图谱改为无横向滚动的静态入口，并新增支持昼夜模式、搜索和 64 卦浏览的知识二级页。
- 将首页与六十四卦知识页的 LightRays / LiquidEther 限制在 Banner 内，并在 Banner 离开视口或页面进入后台后卸载 WebGL。
- 将“今日一卦”卡片按钮改为文案更简洁、点击属性更明确的金色圆角“今日卦象”CTA，并让日期自动对应用户本地年月日。
- 增加逐任务发布确认门禁：后续任务默认停在本地验证和草稿 PR，只有用户明确确认当次发布后才更新 GitHub Pages 公网版本。

## 1.0.0 - 2026-07-21

- 建立根目录唯一源码，将未跟踪的重复项目副本移出工作区并保留可恢复备份。
- 增加项目状态、PRD、路线图、设计系统、架构、组件地图、质量和发布文档。
- 增加自动测试、统一 `npm run check` 与 GitHub 质量工作流。
- 将质量检查与 Pages 发布工作流升级到 Node.js 24 和当前 GitHub Actions 主版本。
- 完成东方观星一级首页的昼夜双主题体验。
- 完成移动端适配、首页书名漂移和逐按钮交互动效。
- 发布 GitHub Pages 公网版本。
- 优化移动端渲染与 Legacy 静态资源加载。
